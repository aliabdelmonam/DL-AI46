# Deep Learning Lab 2

This project is a PyTorch-based machine learning solution tailored to predict the continuous variable `highUptake_mol` from tabular data. It includes data preprocessing pipelines, a fully custom Multi-Layer Perceptron (MLP) architecture, and a robust training loop built from scratch.

## Project Structure

- **`data_preprocessing.py`**: Handles loading data (`train.csv` and `test.csv`), removing corrupt records, and creating generic train/validation splits. Lastly, it transforms pandas DataFrames directly into PyTorch float tensors for efficient computation.
- **`model.py`**: Defines a deep feedforward Multi-Layer Perceptron using PyTorch's `nn.Sequential`. Beyond architecture, the module houses an elaborated, end-to-end custom `fit()` procedure loaded with robust deep learning practices:
  - Weight initializations (Kaiming).
  - Configurable early stopping and regularization.
  - Learning rate scheduling hooks.
  - Gradient clipping capabilities.
  - Model checkpointing to automatically save best and periodic weights during the training process.
- **`main.py`**: The entry point of the project. It glues together data loading utilities with the neural network definitions and defines the hyperparameters (like Adam Optimizer, `StepLR`, MSE Loss, epochs, patience, and regularization coefficients).
- **`checkpoints/`**: Directory autogenerated during training that permanently stores the latest and best performing weights (`.pth` extensions).

## Model Architecture

The core of the prediction model runs on a 3-layer feed-forward Neural Network setup inside the generic `Model` class inherited from PyTorch module.
1. **Input Layer -> Hidden Layer 1**: Maps the shape of input features to 64 neurons.
2. **Activation & Regularization**: Employs `ReLU()` combined with a 50% `Dropout` step to manage network variance and suppress overfitting.
3. **Hidden Layer 1 -> Hidden Layer 2**: Condenses the previous representations down to 32 neurons.
4. **Hidden Layer 2 -> Output**: Compresses the structure into a single linear neuron output mapping to the `highUptake_mol` scale.

## Getting Started

### Prerequisites

Ensure you have python installed in your environment. You will also need to pip install the project dependencies. Ensure that you have `train.csv` and `test.csv` downloaded and located inside the root project directory in order for data ingestion to work.

```bash
pip install -r requirements.txt
```
*(Dependencies normally include: `pandas`, `numpy`, `scikit-learn`, and `torch`)*

### Usage

Proceed to execute the entry script to start preprocessing and training:

```bash
python main.py
```

Expect output monitoring training versus validation metrics iteration by iteration. By the end of the runtime, a final validation score alongside checkpoint records will be present for immediate further inference.
